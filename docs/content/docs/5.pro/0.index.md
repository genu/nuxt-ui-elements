## What is Nuxt Elements Pro?

**Nuxt Elements Pro** extends the free [Nuxt Elements](/docs/getting-started) module with advanced, feature-rich components built for professional applications.

Pro components follow the same design principles as the free library — full TypeScript support, auto-imports, Nuxt UI integration, and dark mode — while providing more complex functionality like calendars, advanced data displays, and more.

::tip{icon="i-lucide-sparkles" title="Early Adopter — Lifetime License"}
Nuxt Elements Pro is in early access. Licensing is **coming soon** — in the meantime, you can install and try out all pro components for free. Early adopters will get a **lifetime license** at a special price, including all future components and updates.
::

- **Support continuous development** — Your license directly funds ongoing work on both the free and pro libraries, keeping the entire ecosystem healthy and actively maintained.
- **Shape the roadmap** — Pro license holders get a direct line to influence feature priorities and request new components. Your feedback drives what gets built next.
- **First-class support** — Get priority responses and dedicated help when you need it.
- **Production-ready components** — Every pro component is built with the same rigor as Nuxt UI itself — full TypeScript support, accessibility, keyboard navigation, and dark mode out of the box.
- **No recurring fees** — Early adopter licenses are lifetime. Pay once, get every future component and update included.

## Installation

Install the pro package alongside the free module:

```bash
pnpm add nuxt-ui-elements-pro
```

Add it to your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  modules: [
    'nuxt-ui-elements',
    'nuxt-ui-elements-pro'
  ]
})
```

## License

Nuxt Elements Pro requires a license key for production builds. Set it via environment variable:

```bash
NUXT_UI_ELEMENTS_PRO_KEY=your-license-key
```

Or in your `nuxt.config.ts`:

```ts
export default defineNuxtConfig({
  uiElementsPro: {
    license: process.env.NUXT_UI_ELEMENTS_PRO_KEY
  }
})
```

During development, the module works without a license key (a warning is logged).

## Configuration

The pro module accepts the same `prefix` option as the free module:

```ts
export default defineNuxtConfig({
  uiElementsPro: {
    prefix: 'UE', // Default. Components: UEEventCalendar, etc.
    license: process.env.NUXT_UI_ELEMENTS_PRO_KEY
  }
})
```
