## Requirements

Before installing Nuxt UI Elements, make sure you have:

- [Nuxt 4](https://nuxt.com) (v4.0.0 or later)
- [Nuxt UI](https://ui.nuxt.com) (v4.0.0 or later)

## Installation

### Step 1: Install the Package

::code-group

```bash [pnpm]
pnpm add nuxt-ui-elements
```

```bash [npm]
npm install nuxt-ui-elements
```

```bash [yarn]
yarn add nuxt-ui-elements
```

```bash [bun]
bun add nuxt-ui-elements
```

::

### Step 2: Add to Nuxt Config

Add `nuxt-ui-elements` to the `modules` section of your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["@nuxt/ui", "nuxt-ui-elements"],
})
```

::note
Make sure `@nuxt/ui` is also included in your modules, as Nuxt UI Elements depends on it.
::

### Step 3: Start Using Components

That's it! Components and composables are now auto-imported and ready to use:

```vue
<script setup lang="ts">
  const dialog = useDialog()

  async function showAlert() {
    await dialog.alert({
      title: "Success!",
      description: "Your changes have been saved.",
      color: "success",
    })
  }
</script>

<template>
  <UButton @click="showAlert"> Show Alert </UButton>
</template>
```

## Configuration

You can customize the module in your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["nuxt-ui-elements"],
  uiElements: {
    // Change the component prefix (default: 'UE')
    prefix: "MyPrefix",
  },
})
```

### Options

| Option   | Type     | Default | Description                             |
| -------- | -------- | ------- | --------------------------------------- |
| `prefix` | `string` | `'UE'`  | The prefix for auto-imported components |

With a custom prefix, components would be used like:

```vue
<template>
  <!-- With prefix: 'MyPrefix' -->
  <MyPrefixDialogAlert title="Hello" />
</template>
```

## TypeScript Support

Nuxt UI Elements is fully typed. Types are automatically available when you install the package:

```ts
import type { DialogAlertProps, DialogConfirmProps } from "nuxt-ui-elements"
```
