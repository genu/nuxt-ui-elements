## Overview

Nuxt UI Elements provides a standard library of utility functions accessible via the `#std` alias. These utilities are tree-shakeable, meaning only the functions you use are included in your bundle.

## Available Utilities

| Utility | Description |
|---------|-------------|
| [plur](/docs/standard-library/plur) | Pluralize strings based on count |
| [slugify](/docs/standard-library/slugify) | Convert strings to URL-friendly slugs |
| [Date](/docs/standard-library/date) | Date manipulation built on @internationalized/date |

## Usage

You can import the entire standard library:

```ts
import std from '#std'

const text = std.plur('item', 5) // "items"
const slug = std.slugify('Hello World') // "hello-world"
const nextMonth = std.date.add(std.date.today(), 1, 'month')
```

Or import specific utilities directly for better tree-shaking:

```ts
import { add, format, today } from '#std/date'

const nextMonth = add(today(), 1, 'month')
const formatted = format(nextMonth, 'MMMM D, YYYY')
```

## Tree-Shaking

Only the functions you import are included in your bundle:

```ts
// Only imports 'format' and 'today' functions
import { format, today } from '#std/date'

format(today(), 'YYYY-MM-DD')
```
